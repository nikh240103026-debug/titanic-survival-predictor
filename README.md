# titanic-survival-predictor

A Machine Learning web app that predicts whether a Titanic passenger would survive.

## Technologies Used
- Python
- Pandas, NumPy, Matplotlib, Seaborn
- Scikit-learn (Random Forest)
- Flask

## How to Run
1. Install requirements: pip install flask scikit-learn pandas numpy
2. Run: python app.py
3. Open: http://127.0.0.1:5000

## Model Accuracy
82.12% accuracy using Random Forest Classifier
